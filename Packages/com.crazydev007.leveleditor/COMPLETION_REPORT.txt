🚀 HOW TO USE

1. OPEN EDITOR
   Unity: Window > Tube Color Level Editor

2. LOAD LEVELS
   Click "Load Levels" button
   Select your color_sort_1000_levels.json file

3. EDIT LEVELS
   Use slider to navigate levels 1-1000
   Edit tubes and properties
   Click "Save Levels" to persist changes

4. VALIDATE
   Tools > Tube Color Level Editor > Validate All Levels
